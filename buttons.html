Default Button
Default Button

Default buttons transition to a lighter color on hover and restore back to their original background when leaving hover.

<div class="ssp-main-container">
  <a class="ssp-btn">Default Button</a>
</div>
.ssp-main-container .ssp-btn {
  display: inline-block;
  margin: 15px 0;
  padding: 13px 60px;
  width: auto;
  vertical-align: middle;

  border: 0 solid transparent;
  border-radius: 0;

  background-image: none;
  background: #41779c;
  color: #fff !important;

  text-align: center;
  font-family: 'proxima_nova_rgbold', sans-serif;
  font-weight: 400;
  font-size: 1em;

  -ms-touch-action: manipulation;
  touch-action: manipulation;
  cursor: pointer;

  -moz-transition: all 0.2s ease-in;
  -o-transition: all 0.2s ease-in;
  -webkit-transition: all 0.2s ease-in;
  transition: all 0.2s ease-in;
}

.ssp-main-container .ssp-btn:hover {
  background-color: #4ba0c8;
  text-decoration: none;
}

.ssp-main-container .ssp-btn:visited {
  text-decoration: none;
}
Hero Button
Hero Button

This button has a transparent background surrounded by a colored border. On hover, it will fill with a specified color.

The ghost-btn class is to be used in addition to the ssp-btn class. Please review the markup below.

<div class="ssp-main-container">
  <a class="ssp-btn ghost-btn">Hero Button</a>
</div>
.ssp-main-container .ssp-btn.ghost-btn {
  padding: 11px 60px;
  border: 2px solid #41779c;
  background: transparent;
  color: #41779c !important;
}

.ssp-main-container .ssp-btn.ghost-btn:hover {
  background-color: #41779c;
  color: #fff !important;
}

.ssp-main-container .ssp-btn.ghost-btn p {
  color: #41779c;
  margin: 0;
  text-align: center;
  font-family: 'proxima_nova_rgbold', sans-serif;
  font-weight: 400;
  font-size: 1em;
}

.ssp-main-container .ssp-btn.ghost-btn:hover p {
  color: #fff;
}
Alert Button
Alert Button

This button is similar to the default button, but instead transitions to a darker color on hover.

The alert-bt class is to be used in addition to the ssp-btn class. Please review the markup below.

<div class="ssp-main-container">
  <a class="ssp-btn alert-bt">Hero Button</a>
</div>
.ssp-main-container .ssp-btn.alert-bt {
  padding: 11px 60px;
  border: 2px solid #ab4501;
  background-color: #d44e05;
  font-family: 'proxima_nova_rgbold', sans-serif;
  font-weight: 400;
}

.ssp-main-container .ssp-btn.alert-bt:hover {
  border: 2px solid #b34304;
  background-color: #b34304;
  color: #fff !important;
}

.ssp-main-container .ssp-btn.alert-bt p {
  font-family: 'proxima_nova_rgbold', sans-serif;
  font-weight: 400;
  font-size: 1em;
}
